
# My Chat App



## Features

- User Authentication
- Sending Messages


## Run Locally

Clone the project

```bash
  git clone https://github.com/arrahman1803/mychatapp.git
```

Go to the project directory

```bash
  cd mychatapp
```

```bash
  cd backend
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python manage.py runserver
```

Add new Terminal and Go to the project directory

```bash
  cd mychatapp
```

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  npm run start
```
